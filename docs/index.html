This is not a compliance system. Applicators must verify official requirements in EPA Bulletins Live! Two for the product, location, and application month.

What The App Does

The Streamlit app highlights cached Alabama-intersecting PULA polygons, checks the nearest cached PULA for a selected point, looks up USDA soil/HSG context, supports ESA mitigation point planning, links users to BLT/PALM, and routes users to Alabama Extension support.

Open App EPA BLT ACES Counties

Current MVP

  • Alabama PULA snapshot loaded from EPA BLT ArcGIS data.
  • USDA Soil Data Access HSG lookup from coordinates.
  • Embedded ESA mitigation point planning worksheet.
  • EDDMapS occurrence-context review panel with radius choices.
  • County office routing from selected coordinates.
  • Crop/site specialist routing from ACES profile data.
  • Private suspected resistance report capture.

Deploying With GitHub Pages

GitHub Pages can host this static project page from the docs/ folder. The Python Streamlit app still needs a Python-capable host such as Streamlit Community Cloud. After that app is deployed, replace the “Open App” link above with the live Streamlit URL.

Data and contacts are attributed to EPA, weedscience.org, Alabama Extension, and Auburn University sources as documented in the project repository.
